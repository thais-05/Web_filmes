📌 Sobre o Projeto

Este projeto foi desenvolvido em 2024 durante o segundo ano do curso técnico de Desenvolvimento de Sistemas da ETEC.

O objetivo principal foi colocar em prática os conhecimentos adquiridos em desenvolvimento Front-End, utilizando tecnologias web para criação de interfaces modernas, responsivas e funcionais.

Durante o desenvolvimento, foram aplicados conceitos de estruturação de páginas, estilização, interatividade com JavaScript e organização de componentes visuais.

🚀 Objetivos do Projeto

Praticar desenvolvimento Front-End;
Criar interfaces responsivas e intuitivas;
Aplicar lógica de programação com JavaScript;
Melhorar organização e estruturação de código;
Desenvolver experiência prática com tecnologias web.

🛠️ Tecnologias Utilizadas

HTML5
CSS3
JavaScript

⚙️ Funcionalidades

Interface responsiva;
Interações com JavaScript;
Navegação dinâmica;
Aplicação de estilização com CSS.
